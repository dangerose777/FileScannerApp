Scanning hash app with save/config file.

## Technologies:
- React
- Vite
- Tauri
- React Icons

## Setup:
in C:\xampp\htdocs\
```
git clone https://github.com/dangerose777/FileScannerApp.git
```
```
cd FileScannerApp-main
```
```
npm install
```

## Run:
```
npm run tauri dev
```
